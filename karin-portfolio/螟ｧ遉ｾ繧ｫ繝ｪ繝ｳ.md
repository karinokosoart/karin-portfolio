## 学歴・バックグラウンド

- 小学校低学年から造形教室に通い、幼少期から美術に親しむ。
- 女子美術大学付属高等学校・女子美術大学でアートを専攻。
- 学生時代からモデル活動と並行してアート制作を続ける。

## 活動内容

### モデル
ファッション誌や広告、ブランドのビジュアルなどで活躍。ナチュラルで個性的な雰囲気を持ち、ストリート／モード系の仕事に多く起用される。

### アーティスト（抽象画）
- 完成イメージを事前に描かず、制作過程で即興的に筆を進めるスタイル。
- 「自分のその瞬間の感情を絵に落とし込む」ことを大切にしており、色とタッチで瞬間の感覚を表現する。
- 原宿のJoint Galleryでの個展「FIGHT」をはじめ、複数の個展を開催。
- キリンビールのCM企画では、2.7mのキャンバスにビールをイメージしたイエロー基調の抽象画6点を制作し話題に。

### 音楽
音楽プロジェクト「**Auraín（オーライン）**」としても活動。絵画に留まらず、音楽という形でも自身の表現を拡張している。

### サステナブルファッション
**GROW&COMPANY LIMITED** とのコラボレーションで、サステナブルファッションアイテムをプロデュース。アート・ファッション・環境意識を横断する取り組みを行っている。

## 思想・スタンス

> 「人間の脳でしか考えられないクリエイトをどうやっていくか」

テクノロジーによる自動化が進む現代において、**人間にしか生み出せない表現や創造性**を追求することに強い問題意識を持つ。即興と感情を軸にした制作姿勢は、この思想を体現している。

## 絵画作品の分析・特徴

### 1. 画風・スタイル
- **抽象画（アブストラクト）** を一貫して選択。具象を避ける理由として、「具体的なものほど言葉で括られてしまう」ことへの違和感を挙げている。
- **見る人の感情との共鳴**で完成する作品観。「見てもらって初めて完成する」と語り、鑑賞者の内面との相互作用を前提にしている。
- 画面全体に自由なストロークと色の重なりが広がり、**即興性・偶発性**が強く表れる。

### 2. 色使い
- アクリル絵の具を主材とし、**鮮やかな発色**と**繊細なかすれ**が共存。
- 単色ではなく、複数の色をキャンバス上で直接混ぜ合わせていくため、**境界が溶け合うグラデーション**や**にじみ**が特徴的。
- キリンビールCMの作品では、**ビールを想起させるイエロー基調**で6点を制作するなど、テーマに合わせて色のトーンを大胆に切り替える柔軟性を持つ。

### 3. 制作プロセス
- **完成イメージを事前に描かない**。先に「使いたい色」をキャンバスに置き、画面で色が混ざり合って生まれる効果から次のイメージを受け取り、手を動かす。
- **感情ドリブン**：その瞬間の自分の感情・気分・心理をそのまま絵に落とし込むスタイル。
- 結果として、同じ作家の作品でも**一枚ごとに表情が大きく異なる**のが特徴。

### 4. モチーフ・着想源
- 夢、感情、瞬間の気分など**内的な体験**を多く題材にする。
- EM(em) AWコレクションで使用された8枚のシリーズでは、**「見た夢」から着想**を得て制作。各作品には個別の感情名／状態名がタイトルとして付けられている。
  - `Upset` / `Dreamy` / `Nocturnal` / `80's breakfast` / `mirror mirror` / `Self hug` / `Wonderer` / `Instant love`
- このタイトリングからも、**感情のスペクトル**（不安／夢想／孤独／ノスタルジー／自愛／恋）を色と形に変換していく作家性がうかがえる。

### 5. タッチ・マチエール
- **力強い筆跡**と**繊細なかすれ**のコントラスト。
- べた塗りではなく、筆の勢いやカスレ、滲みを残すことで **「手の痕跡」＝人間らしさ**を意図的に可視化。
- プリント製品化の際も「アクリルのかすれ具合まで忠実に再現」することが重視されており、**筆致そのものが作品の核**になっていることがわかる。

### 6. 思想的背景
- 「人間の脳でしか考えられないクリエイト」を追求。
- AIやテクノロジーでは代替しきれない **「色の選び方」「タッチの組み合わせ」「ぼかし方」** に価値を置いている。
- その瞬間にしか生まれない偶発性と身体性が、彼女の作品の本質的な魅力になっている。

### 7. 総合的な特徴まとめ

| 観点 | 特徴 |
| --- | --- |
| ジャンル | 抽象表現／感情表現主義的 |
| 色彩 | 鮮やか＋にじみ・かすれ。テーマに応じて大胆に変化 |
| 技法 | アクリル、即興、色の偶発的混色 |
| モチーフ | 夢、感情、瞬間の心理 |
| タッチ | 力強い筆致＋繊細なかすれの共存 |
| 作品観 | 鑑賞者の感情で完成する「開かれた絵」 |
| 哲学 | 人間にしか作れない創造性の探求 |
